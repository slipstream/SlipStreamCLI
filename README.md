# SlipStreamCLI
SlipStream Command Line Interface

This is a command line interface to a SlipStream service.

## Installation
  $ pip install slipstream-cli

## Usage
  $ slipstream --help

## Contribute
  $ git clone https://github.com/slipstream/SlipStreamCLI.git
  $ cd SlipStreamCLI/
  $ pip install --editable .
